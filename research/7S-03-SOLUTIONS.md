**Status**: BACKWASH (retroactive documentation)

## Existing Solutions Analyzed

### 1. Hootsuite / Buffer
- **Pros**: Full-featured, multi-platform
- **Cons**: Expensive, no code-level control

### 2. Mention / Brand24
- **Pros**: Real-time monitoring
- **Cons**: Brand-focused, not developer-oriented

### 3. Custom Scripts (Python)
- **Pros**: Flexible, free
- **Cons**: Maintenance burden, no structure

### 4. IFTTT / Zapier
- **Pros**: No-code automation
- **Cons**: Limited logic, rate limits

## Why Build Custom?

1. **Developer-Focused**: Targets programming communities
2. **Eiffel Integration**: Uses simple_* ecosystem
3. **Draft Workflow**: Human review before posting
4. **AI-Assisted**: Integration points for LLM drafting
5. **Self-Hosted**: No external service dependencies

## Key Differentiators

- SQLite-based local storage
- Multi-platform aggregation
- Draft lifecycle management
- Engagement tracking with reply monitoring
- Resource library for content reuse
